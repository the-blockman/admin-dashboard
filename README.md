# Admin Dashboard

A responsive admin dashboard built with plain HTML and CSS.

## Overview

A front-end recreation of an admin dashboard UI, focusing on CSS Grid and Flexbox layout techniques.

## Features

- Sidebar navigation with icons
- Header with search bar and user profile
- Projects grid with card components
- Announcements and Trending widgets panel

## Technologies Used

- HTML5
- CSS3 (Grid, Flexbox)

## Getting Started

Clone the repository and open `dashboard.html` in your browser. No dependencies or build tools required.
